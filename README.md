```

├── profile.png              ← Your profile photo
└── Vaishnavi_Resume.docx    ← Resume file for download

```

## How to Use

1. Place your `profile.png` and `Vaishnavi_Resume.docx` in the root `portfolio/` folder.
2. Place all certificate images (`cert_1.jpg` through `cert_11.jpg`) inside the `certificates/` folder.
3. Open `index.html` in any browser — everything works locally without a server.

## Assets to Add (not included in ZIP)

These files need to be added by you:
- `profile.png` — your profile photo
- `Vaishnavi_Resume.docx` — your resume
- `certificates/cert_1.jpg` through `certificates/cert_11.jpg` — your certificate images

The website is pixel-perfect identical to the original single-file version.
